# Combination of Differential Evolution (DE) and k-Means for Clustering

🎓 **Project for the Course: Evolutionary Computation / Metaheuristic Algorithms**  
✍️ **Implementation Based on a Scientific and Credible Paper**

---

## 📜 Project Overview
This project presents a combination of two powerful algorithms: **Differential Evolution (DE)** and **k-Means**. The goal of this optimized combination is to perform clustering in a more effective and intelligent way.  
- **Differential Evolution (DE):** A nature-inspired metaheuristic algorithm used for optimization problems.  
- **k-Means Algorithm:** A popular algorithm for data clustering.  

The implementation is based on a scientific paper and aims to improve the performance of data clustering.

---

## 🚀 Features
- ✅ Innovative integration of **DE** and **k-Means** for enhanced clustering.  
- ✅ Optimized and fully functional implementation in MATLAB.  
- ✅ Suitable for data-driven problems and clustering analysis.  
- ✅ Easily extendable and customizable for various datasets and requirements.  

---


## 🧩 Usage
1. Open MATLAB and load the project files.  
2. Run the `main.m` file.  
3. View clustering results and comparisons with other methods in the console and output files.  

---

## 📈 Results
🎯 The project demonstrates that combining DE and k-Means can lead to more accurate and faster clustering, especially for complex and imbalanced datasets.
